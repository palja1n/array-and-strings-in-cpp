## Arrays in C++

# 1. Print Array in Reverse  
**Theory**: Reversing an array means displaying the elements from the last index to the first using a loop.  
**Algorithm**:
- Initialize array with values.
- Loop from last index to first.
- Print each element.
  
Output:

50

40

30

20

10
